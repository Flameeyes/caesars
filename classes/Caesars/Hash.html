Class Caesars::Hash

  1. lib/caesars.rb
Parent: ::Hash

A subclass of ::Hash that provides method names for hash parameters. It’s like a lightweight OpenStruct. 

ch = Caesars::Hash[:tabasco => :lots!]
puts ch.tabasco  # => lots!
