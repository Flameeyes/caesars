Class Caesars::Config

  1. lib/caesars.rb
Parent: Object

A helper for loading a DSL from a config file.

Usage: 

class Staff < Caesars; end;
class StaffConfig < Caesars::Config
  dsl Staff::DSL
end
@config = StaffConfig.new(:path => '/path/2/staff_dsl.rb')
p @config.staff    # => <Staff:0x7ea450 ... >
